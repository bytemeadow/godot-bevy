# Getting Started

This section will walk you through setting up godot-bevy for a new project or an existing one. The new-project path creates a project from scratch and guides you through creating a scene, adding an entity, and controlling it via godot-bevy while the existing-project path guides you through integrating the library and getting one thing controlled by the ECS, and points you to utilities that help with hybrid projects.
