# Getting Started

This section will walk you through setting up godot-bevy for a new project or an existing one. The new-project path creates a project from scratch and guides you through creating a scene, adding an entity, and controlling it via godot-bevy while the existing-project path guides you through integrating the library and getting one thing controlled by the ECS, and points you to utilities that help with hybrid projects.

## System Requirements

For both paths you'll need:

- **Rust** 1.88.0 or later  
    Installation instructions can be found [here](https://rustup.rs/)

- **Godot** 4+  
    We manage our installations with [gdenv](https://gdenv.bytemeadow.com/)

## Getting Started

- **[Creating a New Project](./creating-a-new-project.md)**  
    Create a project from scratch: a scene, an entity, and control it via godot-bevy.

- **[Integrating into an Existing Project](./integrating-into-an-existing-project.md)**  
    Add the library to an existing Godot project and control one node via ECS.
